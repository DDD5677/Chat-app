# Pre-requisites

-  Install [Node.js](https://nodejs.org/en/) version 20.10.0

# Getting started

-  Install dependencies

```
cd backend
npm install
```

-  run the project in dev mode

```
npm run dev
```

Navigate to `http://localhost:3000`

# Frontend

-  Install dependencies

```
cd frontend
npm install
```

-  run the project in dev mode

```
npm run dev
```

Navigate to `http://localhost:5173/`
